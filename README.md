# Library Management System (MERN Stack)

## Tech Stack
- MongoDB
- Express.js
- React (Vite)
- Node.js
- JWT Authentication

## Features
- Admin Authentication
- User Membership Management
- Book CRUD Operations
- Issue & Return System
- Fine Calculation
- Fine Payment
- Overdue Report
- Search & Pagination

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev

Backend runs on: http://localhost:5000  
Frontend runs on: http://localhost:5173
