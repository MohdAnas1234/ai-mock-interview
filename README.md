# AI Mock Interview Platform

AI Mock Interview Platform is a full-stack interview preparation web application where users can practice technical interview questions, track their performance, and receive feedback.

## Features

* User Authentication (Login / Register)
* Protected Routes
* Responsive UI with Bootstrap
* Dark Mode Support
* Interview Categories
* Timer-Based Questions
* Progress Bar
* Result Page with Score Calculation
* Retake Interview Option
* Dashboard Ready Structure
* MERN Stack Ready

## Tech Stack

### Frontend

* React
* Vite
* Bootstrap
* React Router DOM
* Axios

### Backend (Planned)

* Node.js

## Installation

```bash
# Clone repository
git clone https://github.com/yourusername/ai-mock-interview.git

# Move into project folder
cd ai-mock-interview

# Install dependencies
npm install

# Run development server
npm run dev
```
